# Usage

## Daily Flow

1. Open Chabi from tray icon.
2. Add or edit actions.
3. Record shortcuts.
4. Pin key actions (up to 8).
5. Open Quick Bar and execute quickly.

## Core Shortcuts

- Toggle app: `CommandOrControl+Shift+K`
- Toggle quick bar: `CommandOrControl+Shift+L`

## Example Actions

### URL

- `https://github.com`

### App

- `/System/Applications/Utilities/Terminal.app`
- `/Applications/Visual Studio Code.app`

### Bash

```bash
open -na "Visual Studio Code"
open -a "Brave Browser" https://github.com
```
