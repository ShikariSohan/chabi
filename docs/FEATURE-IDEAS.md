# Feature Ideas

This file tracks possible next improvements for Chabi.

## Near-Term
- Editable shortcut presets for common patterns (`Command+Shift+<key>` templates).
- Better action grouping (folders/tags like Work, Coding, Personal).
- One-click duplicate action cloning.
- Quick Bar result badges (type, run status, pin state).
- Reorder pinned actions manually.

## Power-User
- Variables in shell actions (date, clipboard text, selected text).
- Multi-step workflows (run app + open URL + execute shell command).
- Conditional actions (if app is running, do X; otherwise do Y).
- Per-action timeout and retry options for shell commands.

## UX/Design
- Theme presets (glass light, glass dark, high-contrast).
- Compact mode for smaller screens.
- Better empty states and onboarding tips.
- In-app "new feature" changelog panel.

## Platform and Release
- Signed/notarized macOS distribution for smoother install trust.
- Auto-update support.
- Crash/error reporting opt-in.

## Collaboration and Data
- Import from browser bookmarks.
- Sync actions between devices (optional cloud profile).
- Team action packs (share JSON bundles).

## Notes
- Keep additions aligned with `ai-context/` rules.
- Prefer small, testable increments.
