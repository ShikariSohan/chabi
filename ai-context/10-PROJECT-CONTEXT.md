# Project Context

## Mission
Chabi helps users trigger common apps, URLs, shell commands, and system actions quickly from the menu bar.

## UX Direction
- Glassy, lightweight, non-intrusive
- Keyboard-first for quick operation
- Small typography and compact controls

## Current Product Constraints
- Pinned actions are limited to 8
- All actions list is paginated
- Strong input validation for add/edit flows

## Primary Audience
macOS users who want fast automation and app launching without opening a heavy launcher app.
